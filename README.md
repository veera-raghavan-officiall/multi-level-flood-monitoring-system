# Multi-Level Flood Monitoring & Alert System

## Overview
This project simulates a flood monitoring system using Arduino in Tinkercad. It detects water level conditions and categorizes them into safe, warning, and critical stages.

*Tinkercad Link:* [Flood Monitoring System](https://www.tinkercad.com/things/17E7EzdfYUm/editel)

## Components (Simulated)
- Arduino UNO
- Ultrasonic Sensor
- LEDs (Green, Blue, Red)
- Buzzer
- LCD Display

## Working
- Distance is measured using an ultrasonic sensor.
- Based on the distance, the system classifies:
  - ✅ Safe
  - ⚠️ Warning
  - 🚨 Critical Alert
- Alerts are displayed using LEDs, buzzer, and LCD.

## Simulation
This project is fully simulated using Autodesk Tinkercad due to hardware constraints.
